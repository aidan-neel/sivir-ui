# Silk UI

A Svelte 5 + Tailwind CSS v4 component library and docs app.

## Status

This branch is still a work in progress.

## Stack

- Bun
- SvelteKit
- Svelte 5
- Tailwind CSS v4

## Development

The active app lives in [`app/`](/home/aidan/silk/app).

```bash
cd app
bun install
bun run dev
```

Useful commands:

```bash
cd app
bun run check
bun run build
```

## Devcontainer

A Bun-based devcontainer is included in [`.devcontainer/`](/home/aidan/silk/.devcontainer). Opening the repo in a compatible devcontainer client will install dependencies in `app/` automatically.
